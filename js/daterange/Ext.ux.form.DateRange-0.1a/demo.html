Config Options

Entry Description Default
format Sets the format that the dates should be formatted. 'Y-m-d'
delimiter Separator used between two dates. ' to '
handler Callback function to use for handling date selection event.
Example: function ( StartDate, EndDate )		 function(s,e){}
value Value to be used on start-up. Supports DateJs formating.
Example: today-7days		 'today'
presetDates Array of preset objects to display in dropdown menu.
Example: [{ text: 'Caption To Display', dateStart: '...', dateEnd: '...' }, ...]		 see script :)

Public Methods

Method Description Return Value
getDates() Retrieves all the selected date ranges. array
getValue() Retrieves the contents of the text field. string
setValue(v date|string|array) Sets the current value and updates the text field. self
